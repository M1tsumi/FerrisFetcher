# Changelog

All notable changes to FerrisFetcher will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Comprehensive web scraping library with async support
- HTTP client with rate limiting and retry mechanisms
- HTML parsing with CSS selector support
- Configurable data extraction rules and presets
- Concurrent scraping with progress reporting
- Builder pattern for easy configuration
- Comprehensive error handling and types
- Extensive documentation and examples

### Features
- **HTTP Client**: Reqwest-based client with connection pooling
- **HTML Parser**: Robust parsing with malformed content support
- **Data Extractor**: Rule-based extraction with multiple output types
- **Configuration**: Flexible configuration with validation
- **Concurrency**: Tokio-based concurrent request handling
- **Error Handling**: Comprehensive error types and recovery
- **Examples**: Multiple usage examples and comprehensive demo

### Documentation
- Complete API documentation
- Usage examples for all major features
- Contributing guidelines
- Architecture overview

## [0.1.0] - 2024-12-XX

### Added
- Initial release of FerrisFetcher
- Core scraping functionality
- HTTP client with basic retry logic
- HTML parsing with CSS selectors
- Data extraction with configurable rules
- Configuration system
- Error handling framework
- Basic examples and documentation

### Features
- Async HTTP requests with Reqwest
- HTML parsing with Scraper crate
- CSS selector-based element selection
- Configurable extraction rules
- Rate limiting and retry policies
- Builder pattern for configuration
- Progress reporting for batch operations
- Comprehensive error types

### Documentation
- README with quick start guide
- API documentation
- Basic examples
- Project instructions
